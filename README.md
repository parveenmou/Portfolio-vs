# CV Portfolio

## Overview
This project is a personal portfolio website for showcasing skills, experience, and projects. It is designed to be responsive and visually appealing, utilizing modern web technologies.

## Project Structure
```
cv-portfolio
├── index.html          # Main HTML file
├── src
│   ├── css
│   │   └── style.css   # Styles for the portfolio
│   ├── js
│   │   └── script.js    # JavaScript functionality
│   └── assets
│       ├── fonts       # Directory for font files
│       └── svgs        # Directory for SVG files
├── .gitignore          # Git ignore file
├── package.json        # npm configuration file
└── README.md           # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd cv-portfolio
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```

## Usage
- Open `index.html` in a web browser to view the portfolio.
- Modify the `src/css/style.css` file to change styles.
- Update `src/js/script.js` for any interactive features.
